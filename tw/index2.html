蔚藍海洋｜三重
蔚藍海洋｜三重
蔚藍海洋｜三重

坐擁廣大且蔚藍海洋的三重縣
逃離繁華都市的擁擠人潮

享受三重天然海味。體驗在地人文風情，帶著父母一同感受海洋的慰藉

該路線旅遊路線圖
第一天

: 中部國際機場 → 松阪木棉和服體驗 → 御城番屋敷 → 和田金

第二天

: 松阪市 → 伊勢神宮 → 托福橫丁 → 鳥羽市

第三天

: 鳥羽市 → 伊勢志摩Skyline → 海女小屋Satoumian → 志摩市

第四天

: 志摩市 → Mikimoto 珍珠島→ 津市

搭船

: 津市 → 中部國際機場

路線名稱：蔚藍海洋｜三重

新特麗亞名古屋中部國際機場
A

到了新特麗亞名古屋中部國際機場！從入境口可以很順暢地移動到租車區域。帶父母旅遊，租車自駕是最好的選擇！
1小時45分鐘

松阪木綿和服體驗
B

相關資訊：

住所：〒515-0084 三重縣松阪市日野町589-1

営業時間：11:00 ~18:00

江戸時代（17～19世紀）興起的松阪木棉和服，藍染色調與格紋至今仍非常受歡迎。孝親旅遊絕對不能錯過歷史文化體驗之一！

3分鐘

☆雨天備案☆松阪木棉手織中心
C

相關資訊：

營業時間：9:00~17:00

費用：日幣1,300圓

HP URL： http://matsusakamomen.com/

松阪木棉手織中心是江戶富豪三井家的舊址。這裡販賣著松阪木棉的服飾及小物等，還可以體驗傳統織布。稀有的文化體驗，可以與父母一同創造美好的回憶！
1分鐘

和田金
D

相關資訊：

住址：〒515-8588　三重縣松阪市中町1878番地

營業時間：11:30~20:00

費用：12,900圓~

HP URL：http://e-wadakin.co.jp/menu/

相關影片：https://www.facebook.com/145611475834905/videos/2171007676541529/

和田金可以說是松阪牛料理的顏質擔當。創業於明治初年，現在可說是最具代表性的松阪牛名店。用高級的料理，回饋父母長久以來的辛勞！高品質的待客服務，也能讓父母受到被款待的喜悅。實現孝親旅遊的意義。
5分鐘

志摩市温泉旅館
A
同樣擁有許多天然溫泉的志摩地區，風景與鳥羽地區則大有不同。入住志摩的溫泉旅館，放眼望去可以看到由眾多個小島刻畫、排列出的英虞灣風景，感受自然與海洋交織的美。
